Project Subject: the Board Game Club & Resource Hub

Project Description:

The website will serve as an informational and community space for board game enthusiasts. It will include game reviews, strategy guides, tutorials for beginners, recommendations by game type (party, strategy, cooperative, etc.), a calendar of local meetups or events, and a blog where members can share experiences or game night recaps. It may also feature spotlights on unique or indie games.
